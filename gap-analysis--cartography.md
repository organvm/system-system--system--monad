  ---
  Gap Analysis: The Ontological Cartography as One Instance of Its Own Laws

  The cartography is beautiful and self-referential in the narrow sense (the diagram's existence obeys the existence condition it diagrams). But
   it commits six structural errors when measured against the formal system it claims to inhabit.

  Gap 1: Wrong Ontology Being Mapped

  The cartography maps V(Π) — plane-existence variables from physics (dimensionality, boundary, force geometry, temporality, energy, interaction
   rules, phase spectrum). This is a domain-specific instantiation. The system's own Law 3 (Primitive Stratification) says primitives are:
  distinction, token, relation, rule, flow, feedback, governance. The cartography should be able to render either ontology — or better, the
  meta-schema from which both derive.

  The system has no self-cartography. It maps a physics plane but cannot map itself.

  Gap 2: Scale Invariance Not Demonstrated Visually

  Law 4 says the manifest schema is a fixed point — identical at every σ. The cartography is locked at σ = 0. There is no
  zoom-into-a-node-and-find-the-same-structure behavior. The .md essay claims "zoom into a module and its primitives become visible" but the
  HTML doesn't implement it. The primitives orbit as decorative dots — they are not clickable sub-manifests.

  The self-similarity is asserted, not enacted.

  Gap 3: The Containment Stack Is Absent

  The governance framework (ATM-G-001 through G-012) defines a 10-tier containment stack: Physical Law → Info-Theoretic → Chemical → Planetary →
   Biological → Temporal → ... → Human₂₀₂₆. This is arguably the most ambitious structural claim in the system. The cartography has an
  "existence condition ring" — a single golden circle — but nothing that renders the nested containment hierarchy.

  The outermost governance layer has no visual representation.

  Gap 4: The Seven Parallel Studies Are Invisible

  The derivation parallel-studies--seven.md (ATM-P-001 through P-010) frames the primitives not as settled layers but as seven live inquiries
  running simultaneously: Signal Theory, State Mechanics, Rule Dynamics, Interface Negotiation, Loop Ecology, Schema Competition, Protocol
  Genesis. The system is their interference pattern. The cartography has no mechanism to show competing concurrent processes converging.

  The generative engine of the system is not cartographed.

  Gap 5: No Observer Position

  ATM-S-002 declares the observer stance: "eyes fixed across temporal modes (was, is, becoming) — atemporal." The cartography has no observer.
  It has a viewport, but no formal position within the system from which observation occurs. Law 4 says observation is peer-to-peer — you
  observe from an adjacent entity at the same scale, never from outside. The current map is god's-eye, not peer-to-peer.

  The cartography violates its own observation axiom.

  Gap 6: The Entropy Measure Is Decorative

  Law 1 is the most fundamental: H(S) < Σ H(e_i) ⟺ S is a system. The cartography claims self-reference because the diagram's existence depends
  on the four-part conjunction Φ > 0 ∧ |Σ| ≥ 2 ∧ Τ.dir ≠ ∅ ∧ Ι ≠ ∅. But this is a domain-specific existence condition for physics planes, not
  the master law. The actual test would be: does the diagram's joint entropy fall below the sum of its component entropies? If you could disable
   nodes and measure the resulting loss of systemic coherence (mutual information), that would be a genuine enactment of Law 1. Currently the
  existence ring is a label, not a computation.

  The defining law of the system is invoked but not computed.

  What the Cartography Needs to Become

  Three possible directions, from narrowest fix to most ambitious:

  A. Parameterize the Ontology (Minimum Fix)

  Make NODE_DEFS and EDGE_DEFS loadable from external data. Supply the V(Π) physics ontology as one dataset and the 7-layer primitive
  stratification as another. The force model, rendering, and flow logic remain unchanged — only the topology changes. This is the cheapest move
  and immediately demonstrates that the cartography is schema-agnostic.

  B. Implement Scale Descent (Law 4 Enactment)

  When a node has composition.type = "system", clicking it replaces the graph with the sub-manifest at σ-1. The same force model runs on the
  inner graph. Breadcrumbs show the scale stack. This proves Law 4 operationally — the same rendering engine, the same schema, at every level.

  C. Self-Cartography (The Full Recursive Object)

  The cartography renders its own formal system: the 4 Laws as containers, the 5 axiom documents as modules, the 4 derivations as emergent
  nodes. The edges are the actual dependency relations from atom-registry.yaml. The 67 atoms are the primitives orbiting within their parent
  documents. The containment stack is the existence ring — but now with nested rings, one per governance tier. The seven parallel studies
  animate as concurrent wave-fronts rather than particle flows.

  At this level, the map literally is the territory: modifying a node in the cartography would modify the formal system, and modifying the
  formal system would re-render the cartography. The self-referential closure ATM-M-004 describes — "the spec is a configuration field in a
  materia of concepts; reading changes the reader" — would become operational.
